## 2. Pertanyaan Inti

**Bagaimana karya cultivation berbeda dalam merancang bakat, spiritual roots, physiques, dan sistem ketimpangan akses menuju kekuatan?**

---

## 3. Model Besar

### Deterministic Root Model
Bakat awal sangat menentukan nasib.

### Flexible Aptitude Model
Bakat penting, tetapi pilihan, teknik, dan resource masih bisa mengubah arah.

### Constitution-Heavy Model
Tubuh spesial lebih dominan daripada root.

### Hidden/Awakening Model
Potensi sebenarnya tidak terbaca sejak awal atau bangun di tengah perjalanan.

---

## 4. Implikasi Komparatif

Model bakat menentukan:
- seberapa keras aristokrasi spiritual dunia;
- apakah underdog trope masih masuk akal;
- bagaimana sekte memilih murid;
- seberapa penting bloodline politics.
