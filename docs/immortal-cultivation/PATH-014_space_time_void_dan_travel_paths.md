## 2. Pertanyaan Inti

**Bagaimana jalur space/time/void/travel bekerja dalam cultivation, dan apa yang terjadi ketika mobilitas, perpindahan, atau manipulasi ruang-waktu dijadikan pusat kekuatan?**

---

## 3. Signifikansi Topik

Jalur ini penting karena ia memindahkan cultivation dari fokus pada “kekuatan menyerang” ke fokus pada:
- posisi,
- jarak,
- momen,
- arah,
- keterputusan,
- dan keberadaan antar-ruang.

Di banyak dunia, karakter dengan jalur ini dapat:
- lolos dari lawan jauh lebih kuat,
- membuka rute baru,
- memanfaatkan secret realms,
- atau merusak struktur dunia itu sendiri.

---

## 4. Batas File

File ini membahas:
- path yang memakai ruang, waktu, kehampaan, atau travel sebagai pusat progression;
- aplikasi praktis dan identitas jalurnya.

File ini tidak membahas secara penuh:
- ontologi ruang-waktu (`CORE-008`);
- temporal experience dan nostalgia (`CORE-013`);
- transportasi sebagai infrastruktur sosial (`WORLD-013`).

---

## 5. Definisi Kerja

Space/time/void/travel path adalah jalur cultivation yang memusatkan diri pada **manipulasi hubungan spasial, temporal, kehampaan, dan perpindahan** sebagai sumber kekuatan, strategi, dan transendensi.

---

## 6. Space Path

Space path biasanya mencakup:
- teleportation,
- spatial cutting,
- pocket-space creation,
- distance manipulation,
- battlefield folding.

Kekuatan utamanya adalah mengubah relasi “di mana sesuatu berada”.

---

## 7. Time Path

Time path biasanya mencakup:
- acceleration,
- slowing,
- delay,
- temporal perception shifts,
- windows of causality.

Time path sangat berbahaya karena bahkan manipulasi kecil terhadap waktu dapat mengubah:
- duel,
- healing,
- aging,
- and strategic planning.

---

## 8. Void Path

Void path berkaitan dengan:
- ketiadaan,
- ruang kosong,
- interstitial worlds,
- non-place,
- devouring gap between structures.

Ini sering menjadi jalur yang lebih abstrak, lebih berbahaya, dan lebih dekat ke kosmologi tinggi.

---

## 9. Travel Path

Travel path lebih fokus pada:
- movement,
- crossing,
- traversing hostile realms,
- pathfinding between worlds,
- route mastery.

Ia bisa bersinggungan dengan space/time path, tetapi tidak harus identik. Travel path lebih dekat ke **mobilitas eksistensial**.

---

## 10. Mengapa Jalur Ini Kuat

1. mengubah positioning dan access
2. membuka rute yang orang lain tidak bisa tempuh
3. memengaruhi eksplorasi dan escape lebih dari frontal combat
4. sangat terkait dengan kosmologi, secret realms, dan ascension
5. dapat mengubah rasa seluruh dunia bila dikuasai institusi tertentu

---

## 11. Keterbatasan Jalur Ini

1. sangat sulit dikontrol
2. risiko backlash kosmologis besar
3. path ini mudah merusak konsistensi dunia jika terlalu bebas
4. banyak teknik space/time harus diberi batas keras agar cerita tetap tegang

---

## 12. Risiko Khusus

### 12.1 Spatial dislocation
Tubuh, jiwa, atau artifact terpotong dari posisi benar.

### 12.2 Temporal mismatch
Karakter tertinggal zaman, menua tak seimbang, atau hidup di bawah ritme waktu yang salah.

### 12.3 Void contamination
Kehampaan tidak hanya kosong, tetapi bisa mengikis identitas dan struktur tubuh.

### 12.4 Route dependency
Pengguna travel path dapat sangat tergantung pada jalur, node, atau kondisi kosmik tertentu.

---

## 13. Variasi antar Subgenre

### Classical cultivation
Lebih terbatas, biasanya terkait teleport arrays, hidden travel arts, atau escapist techniques.

### Xuanhuan battle-heavy
Space/time path bisa menjadi combat spectacle dan high-level law control.

### Dark cultivation
Void path bisa sangat dekat dengan annihilation, madness, non-being, or reality erosion.

### Sci-fi/interstellar
Jalur ini dapat menjadi tulang punggung mobilitas antarbintang, warp-like traversal, atau void highway mastery.

### System cultivation
Dapat diterjemahkan sebagai route unlock, fold-space skill, or temporal instance mechanics.

---

## 14. Klise yang Perlu Diwaspadai

1. teleportasi tanpa biaya
2. manipulasi waktu tanpa konsekuensi
3. void hanya efek visual hitam-ungu
4. path ini menjadi solusi semua masalah secara instan

---

## 15. Peluang Inovasi

### 15.1 Route sovereignty
Siapa menguasai jalur antarrealm menguasai perdagangan dan perang.

### 15.2 Time scars
Pengguna time path menanggung retak temporal pada identitas dan relasi sosial.

### 15.3 Void law as anti-world
Void bukan elemen, tetapi law anti-struktur yang memakan bentuk.

### 15.4 Travel path as civilization-builder
Jalur ini tidak hanya untuk kabur, tetapi untuk membuka jaringan dunia baru.
