## 2. Tujuan Template

Template ini dipakai untuk merancang path yang:
- punya medium jelas,
- punya harga dan blind spot,
- punya efek terhadap masyarakat,
- dan terasa berbeda dari path lain.

---

## 3. Pertanyaan Inti Desain

1. Apa inti jalan ini?
2. Apa medium utamanya: tubuh, qi, darah, suara, tulisan, faith, beast, time, dsb.?
3. Apa jenis orang yang cenderung menempuh path ini?
4. Apa path ini berikan yang path lain tidak?
5. Apa harga khasnya?
6. Bagaimana path ini mengubah institusi dan konflik dunia?

---

## 4. Template Path Sheet

- **Nama path**
- **Definisi pendek**
- **Medium utama**
- **Kompatibilitas** (roots/physiques/temperament)
- **Teknik khas**
- **Kelebihan**
- **Blind spot**
- **Harga / risiko**
- **Prestise sosial**
- **Resource utama**
- **Bentuk institusi yang cocok**
- **Tone yang paling cocok**

---

## 5. Failure Modes yang Harus Dicek

1. path hanya beda efek visual
2. semua path tetap menyelesaikan konflik dengan cara identik
3. path tidak memengaruhi ekonomi, budaya, atau institusi
4. path tidak punya harga khas

---

## 6. Mini-Contoh Pemakaian

**Nama path:** Archive-Burning Path

- medium: teks, abu prasasti, dan memori hukum
- kompatibilitas: literate cultivators with strong memory discipline
- kelebihan: dapat membongkar seal dan membaca law remnant
- blind spot: lemah di duel frontal dan rentan memory overload
- harga: kehilangan sebagian memori personal tiap kali membuka scripture terlalu dalam
- institusi cocok: ruined libraries, legal sects, underworld record bureaus
