## 2. Tujuan Dokumen

Dokumen ini berfungsi sebagai register pusat untuk seluruh arsip riset **Immortal Cultivation / Xianxia / Xiuzhen**. Fungsi utamanya meliputi:

1. mencatat seluruh file yang direncanakan;
2. menetapkan status pengerjaan setiap file;
3. menandai prioritas implementasi;
4. menghubungkan file dengan milestone proyek;
5. menjadi titik masuk administratif bagi seluruh arsip.

Dokumen ini tidak memuat pembahasan substantif topik cultivation. Seluruh pembahasan konseptual berada pada file domain lain.

---

## 3. Legenda Status

### 3.1 Status pengerjaan
- **Not Started** → belum ditulis
- **In Progress** → sedang ditulis / disusun
- **Draft Ready** → draft pertama selesai
- **Reviewing** → sedang direviu / dicek overlap / dicek referensi
- **Stable** → relatif final untuk versi berjalan
- **Living** → aktif dan dijadwalkan diperbarui berkala

### 3.2 Prioritas
- **P1** → fondasi langsung untuk file lain
- **P2** → penting, tetapi bergantung pada fondasi P1
- **P3** → ekspansi tematik
- **P4** → satelit / pelengkap / spesialis

### 3.3 Level target
- **L1** → ringkas operasional
- **L2** → mendalam
- **L3** → arsip inti / rujukan utama
