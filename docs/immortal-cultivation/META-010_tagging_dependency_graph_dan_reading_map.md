## 3. Ringkasan

Dokumen ini mengatur tag tematik, relasi prasyarat, dan jalur baca per tujuan. Tag minimum mencakup: #ontology #progression #world #market #toolkit. Dependency graph dipakai untuk menunjukkan file owner dan file turunan. Reading map memecah arsip menjadi jalur Worldbuilder, Genre Analyst, Foundations, Power System, dan Market Reader.
