## 2. Pertanyaan Inti

**Bagaimana prestige, awards, influence lists, dan nilai IP membantu menentukan karya cultivation mana yang bertahan sebagai nama besar lintas waktu?**

---

## 3. Signifikansi Topik

Tidak semua karya besar diukur oleh ranking mingguan. Beberapa bertahan melalui:
- adaptasi lintas media,
- prestige author,
- continued reprints,
- awards,
- dan posisi kuat pada influence/IP lists.

File ini penting untuk memisahkan “ramai” dari “bertahan”.

---

## 4. Fungsi Analitis

- menghubungkan popularitas dengan prestige;
- membaca bagaimana karya bergerak dari serial panas menjadi IP besar;
- menilai kenapa karya tertentu tetap hidup meski tidak selalu sedang trending.
