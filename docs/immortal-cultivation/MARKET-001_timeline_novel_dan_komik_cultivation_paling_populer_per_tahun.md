## 2. Fungsi Dokumen

File ini berfungsi sebagai arsip temporal untuk:
- karya cultivation yang dominan pada periode tertentu;

---

## 3. Kerangka Periodisasi Awal

### Fase awal / fondasional
Ciri:
- lahirnya webnovel besar,
- pencarian bentuk xianxia modern,
- pengaruh kuat karya-karya pionir.

### Fase ekspansi platform
Ciri:
- serial panjang makin dominan,
- worldbuilding dan progression ladder mengeras,
- beberapa karya menjadi standar baru.

### Fase IP dan adaptasi besar
Ciri:
- donghua/manhua/drama memperbesar jangkauan,
- karya populer tidak lagi hanya novel,
- ranking dan IP value mulai menjadi indikator paralel.

### Fase hybrid modern
Ciri:
- comedy/meta,
- dark cultivation,
- system cultivation,
- urban dan sci-fi hybrids meningkat.

---

## 4. Fungsi Analitis

Timeline ini tidak hanya menanyakan “apa yang populer”, tetapi juga:
- model cultivation apa yang menang pada era tertentu;
- tipe protagonis apa yang dominan;
- apakah pembaca mengarah ke drama emosional, battle-heavy escalation, atau hybrid systems.
