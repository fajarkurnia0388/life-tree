# Rencana Topik Riset Terpisah: Immortal Cultivation / Xianxia / Xiuzhen

Dokumen ini adalah **peta riset** untuk memecah topik besar *immortal cultivation* menjadi file-file terpisah dan mendalam.

Tujuan utamanya:
1. membangun **arsip riset modular**, 
2. memudahkan pendalaman per topik,
3. menjaga agar tiap file punya fokus jelas,
4. memungkinkan perbandingan antar jenis cerita cultivation.

---

## Prinsip penyusunan file

Setiap file riset nantinya idealnya memuat bagian-bagian berikut:
- definisi inti,
- akar filosofis / mitologis,
- istilah penting,
- variasi antar subgenre atau jenis cerita,
- fungsi dalam worldbuilding,
- fungsi dalam sistem kekuatan,
- efek pada karakter, masyarakat, dan konflik,
- trope umum,
- peluang inovasi untuk setting original.
